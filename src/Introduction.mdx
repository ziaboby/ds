The _UI_ folder includes the components styles, while the _Pattern_ one the stateful logic.

Stories are collected under a _stories_ folder since there are two versions, one for each device macro-type.

CSS modules are used to implement styles using unique class names while keeping low the rules specificity.
Each module is added at component level, while the global styles and the CSS properties are collected in _src/index.css_.
